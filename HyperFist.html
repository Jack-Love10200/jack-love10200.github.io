HYPERFIST - Implementation Highlights

HYPERFIST is a fast-pased first person beat-em-up that I joined halfway through development to help with graphics work. It leans heavily on an over-the-top comic book/cartoon art style, bold colors, sharp silhouettes and exaggerated animations. My main contributions on this project were implementing impact frames to enhance the feel of big punches, adding support for steam features (achievements, leaderboards) as well as preparing the game for release on Steam Deck.
